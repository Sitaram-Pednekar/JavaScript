üß† What is JavaScript? (Theory)

JavaScript is a high-level, interpreted, dynamic, prototype-based scripting language mainly used to make web pages interactive.

Key theoretical characteristics:

Concept	Meaning in JavaScript
High-level	Easy for humans to read/write; abstracts memory management
Interpreted / JIT	Runs directly in the browser/engine without traditional compilation (modern engines use Just-In-Time compilation)
Dynamic typing	Variable types are decided at runtime
Prototype-based OOP	Objects inherit from other objects via prototypes (not classical classes originally)
Event-driven	Code often runs in response to user or system events
Single-threaded with event loop	Uses an event loop + callbacks/promises/async to handle concurrency
Multi-paradigm	Supports procedural, object-oriented, and functional programming
First-class functions	Functions can be stored in variables, passed as arguments, returned from functions

JavaScript runs in:

Browsers (Chrome V8, Firefox SpiderMonkey, Safari JavaScriptCore)

Servers (Node.js)

Mobile apps, desktop apps, IoT, games (via various frameworks)

‚ùå Disadvantages of JavaScript
Disadvantage	Explanation

Browser inconsistencies	Different browsers may behave slightly differently
